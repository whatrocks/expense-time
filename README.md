# expense-time

<img src="./et.png">

>Expenses. Excitement. A Jedi craves not these things.

Expense Time is a React Native iOS app to view and create expenses using the Expensify API.

## Requirements
- react-native: 0.19.0
- redux: 3.1.7
- react-redux: 4.1.2
- redux-thunk: 1.0.3

## Usage
- clone the repo

```bash
cd expenseTime
npm3 install
```
- build and run the project in xcode
